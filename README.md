# Springboot人事管理系统 java536

## 项目概述

本项目是一款基于JavaWeb技术的人事管理系统，采用前后端分离的架构设计。后端采用Springboot框架，前端使用Vue框架。该系统适合作为学习与练手的实践项目，有助于深入理解企业级应用的开发流程和设计理念。

## 技术栈

- **后端：**
  - Spring Boot
  - Spring Framework
  - MyBatis
  - MySQL
- **前端：**
  - Vue.js
  - Element UI
  - Axios
- **其他：**
  - Maven
  - Git

## 功能模块

系统主要包含以下功能模块：

- 用户管理
  - 用户登录与登出
  - 用户权限分配
- 员工管理
  - 新增/编辑/删除员工信息
  - 员工信息查询
- 部门管理
  - 新增/编辑/删除部门
  - 部门信息查询
- 薪资管理
  - 薪资结构设置
  - 员工薪资记录管理

## 系统角色

- 系统管理员：负责系统的维护和用户权限管理。
- 普通用户：进行日常的人事管理操作。

## 运行环境

- **后端：**
  - JDK 1.8 或更高版本
  - MySQL 5.7 或更高版本
- **前端：**
  - 现代浏览器（Chrome、Firefox等）

## 部署步骤

1. 克隆项目代码到本地
   ```bash
   git clone [项目地址]
   ```
2. 进入项目目录，使用Maven构建项目
   ```bash
   mvn clean package
   ```
3. 部署后端服务
   - 配置数据库连接信息
   - 部署到Java应用服务器（如Tomcat）
4. 部署前端项目
   - 进入前端项目目录
   - 安装依赖
     ```bash
     npm install
     ```
   - 编译并启动前端服务
     ```bash
     npm run serve
     ```
